## How to Run Locally

If you want to check the code on your local machine:

1.  **Clone the repo:**
    ```bash
    git clone [https://github.com/techiviju/Portfolio.git](https://github.com/techiviju/Portfolio.git)
    ```
2.  **Navigate to the folder:**
    ```bash
    cd Portfolio
    ```
3.  **Install dependencies:**
    ```bash
    npm install
    ```
4.  **Start the server:**
    ```bash
    npm start
    ```
